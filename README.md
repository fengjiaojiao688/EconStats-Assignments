# Payment Willingness Prediction System based on Random Forest
Developed a comprehensive machine learning prediction system to analyze consumer behavior in the pet funeral market. The project utilizes Python (Pandas, Scikit-learn) to process mixed data types (categorical and continuous variables) and constructs a Random Forest Regression model, optimized via GridSearchCV. It implements a full suite of model interpretability analyses, including residual analysis, feature importance ranking, permutation importance validation, and Partial Dependence Plot (PDP) visualization. Finally, a complete Pipeline including the scaler is outputted for future deployment.
